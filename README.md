 Project 1: Dice Rolling Game
 Description:

This program simulates rolling a pair of dice.
Each time the program runs, it generates two random numbers between 1 and 6 (inclusive), representing the result of each die.

The program displays the result and asks the user if they want to roll again.

Features:

1)Generates two random numbers (1–6)

2)Displays dice results

3)Allows user to roll again

4)Uses loop for repetition

Concepts Used:

Random number generation (rand() and srand())

Loops (do-while)

Conditional statements

User input/output

 Project 2: To-Do List Application
Description:

This is a simple console-based To-Do List application.
It allows users to add tasks, view tasks, and remove tasks once completed.

Features:

1)Add new tasks

2)View all current tasks

3)Remove completed tasks

4)Menu-driven interface

 Concepts Used:

*Vectors (Dynamic Array)

*Loops

*Switch statement

Functions of STL (push_back, erase)

User input/output
